# CodeBlock Wrapper
CodeBlock Wrapper can take following parameters:
- `language`: Define syntax coloration depending on language (\```ts ...```)
- `header`: Hide HeaderBlock when value set to `"false"` (`header="false"`)
- `title`: Set a title in HeaderBlock (`title="My Title"`)
- `flag` : Set a flag in HeaderBlock, override calculated flag (`flag="My Flag"`)
- `icon` : Set an icon in HeaderBlock, override calculated icon (`icon="ðŸŽ–ï¸"`)
- `useIcon` : Hide Icon in HeaderBlock when value set `"false"` (`useIcon="false"`)
- `useFlag` : Hide Icon in HeaderBlock when value set `"false"` (`useFlag="false"`)


## Demo
### Simple CodeBlock
No HeaderBlock by default
> \`\`\` ... \`\`\`
```
{
    href: 'https://github.com/XanTiuM-Dev',
    // label: 'GitHub',
    position: 'right',
    className: 'header-icon header-github-link',
},
```


### CodeBlock + language
HeaderBlock **with language name as flag** and **language icon as icon**
> \`\`\`ts ... \`\`\`
```ts

    // label: 'GitHub',
    position: 'right',
    className: 'header-icon header-github-link',

```
